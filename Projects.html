Technical Projects

2048 Game Solver | Monte Carlo Tree Search

Engineered an automated solver for the game 2048 utilizing a Monte Carlo Tree Search (MCTS) algorithm. The system performs thousands of random simulations (playouts) per move to evaluate potential board states, using the results to select the optimal direction. I implemented a custom scoring heuristic that prioritizes tile merging and corner-heavy board configurations, significantly increasing the probability of reaching the 2048 tile compared to standard greedy algorithms.

Predictive Grade Analysis | Machine Learning

Developed a machine learning model to predict student academic outcomes by implementing Perceptrons from scratch. I conducted a comparative analysis between Linear Regression (for continuous GPA prediction) and Classification (for letter grade categorization) methods. The project involved extensive data preprocessing and feature engineering to evaluate model accuracy, culminating in a report detailing the strengths of each method regarding convergence rates and error margins.

Carputer Driving Tracker | IoT & Geospatial Systems

Currently architecting a vehicle-integrated tracking system powered by a Raspberry Pi 4. The system leverages an RTL-SDR (Software Defined Radio) and GPS modules to provide real-time spatial awareness. The backend utilizes PostgreSQL with PostGIS extensions to store and query complex trajectory data.

Key Features:
  • Real-time Telemetry: Instantaneous "Where am I?" tracking and live trip logging.
  • Historical Analytics: Users can visualize past tracks and calculate road coverage using geospatial queries.
  • Containerized Architecture: Managed via Docker Compose to ensure seamless deployment across hardware environments.
