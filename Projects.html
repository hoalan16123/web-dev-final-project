Developed as part of a high-performance team, this project features an automated solver for the game 2048. I co-engineered the core game engine and the Monte Carlo Tree Search (MCTS) implementation.

While the team explored various strategies, I specifically developed and optimized a custom scoring heuristic that prioritized "Corner-Weighting." By rewarding board states where high-value tiles remained in the corners, my implementation significantly outperformed greedy algorithms, and achieved a tile of 2048.

Technical Breakdown:
  • Game Logic: A 4x4 grid simulation using sliding/merging mechanics.
  • MCTS Agent: For every move, the agent executes thousands of random simulations (playouts) to map out potential future states.
  • Decision Making: The algorithm selects the move with the highest average score/win-rate based on the simulated outcomes.

Developed as a collaborative research project, this study utilized a Multi-Layer Perceptron (MLP) Neural Network> to predict student academic performance based on a diverse dataset of socio-economic and behavioral features, including study habits, parental education, and demographic data.

I architected and implemented the Regression Model, designed to predict a continuous GPA value (e.g., 3.7) rather than a fixed category. I performed a comparative analysis between my regression approach and the team's classification model (letter grades). My findings demonstrated that the regression model provided a higher degree of predictive precision by capturing the subtle variances in the data that are often lost during the "binning" process of classification.

Technical Stack:
  • Architecture: Multi-Layer Perceptron (Neural Network) with tuned hidden layers.
  • Input Features: Age, Gender, Ethnicity, Parental Education, and Weekly Study Time.
  • Evaluation Metrics: Analyzed model performance using Mean Squared Error (MSE) for regression and Accuracy/F1-Score for classification.
