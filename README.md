### Webdev Final Project

This project is a simple web application built using HTML, CSS, and JavaScript.
It does not rely on any external libraries or frameworks, making it lightweight and easy to run.

# Features:

    - Collapsible sections for organized content
    - CSS transformations for visual effects
    - Animations for improved user experience
    - Modal popups for interactive elements
    - Multi-page navigation structure
    - Use of HTML data attributes for dynamic behavior

# Getting Started

    Prerequisites

        - A web browser (Chrome, Edge, Firefox, etc.)
        - A live server (recommended)

# Running the Project

    1. Open the project folder in your code editor (e.g., VS Code).

    2. Right-click on the main HTML file (e.g., index.html).

    3. Select "Open with Live Server".

    4. The project will launch in your default browser.

        Note: No installation or dependencies are required.
